Python Practice Basic - W3Resource

This repository contains Python exercises from https://www.w3resource.com along with my attempted solutions. 
It is designed for anyone who wants to practice and improve their Python programming skills.

Contents:
- Basic Python exercises
- Data types and operations
- Control structures (if-else, loops)
- Functions and modules
- Object-oriented programming exercises
- Miscellaneous Python challenges

How to Use:
1. Clone the repository:
   git clone https://github.com/yourusername/python-practice-basic-w3resource.git
2. Navigate to the relevant folder and open the Python files in your editor.
3. Run the scripts using Python 3:
   python3 filename.py
